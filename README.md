# Campsite Commander 🏕️

## What is this app? ##
Campsite Commander is an Android app that helps campers 
track their gear and food supplies before heading out.

## Features ##
- Splash screen with app branding
- Main screen showing total items packed
- Detailed gear list view with categories and notes
- Dark mode nature themed UI

## Screens ##
1. Splash Screen - displays logo for 3 seconds
2. Gear Screen - shows total items calculated using a loop
3. Detail Screen - displays full gear list from parallel arrays

## How to run ##
1. Open in Android Studio
2. Run on emulator or physical device
3. App launches automatically

## Pseudocode ##
- Initialise parallel arrays with gear data
- Loop through quantities array to calculate total
- Display total on main screen
- Pass arrays via Intent to detail screen
- Loop through arrays to display each item
